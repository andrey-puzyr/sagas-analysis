Программа выполняет следующее:
- анализирует текст, указанный по Интернет-ссылке / URL
- находит числительные по словарю вместе с контекстом (одно предложение, где встречается это числительное)
- выводит найденные соответствия в виде таблицы

Формат итоговой таблицы по колонкам:
1. номер по порядку
2. номер главы (это римская цифра в начале строки, встречающася последний раз до найденного соответствия)
3. конкретное найденное число
4. полное предложение, в которой число было найдено

Программа должна быть в виде html-файла с логикой на JavaScript, функционал должен быть покрыт тестами.
